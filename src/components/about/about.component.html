About the Project

Cubel Cloud is a comprehensive demonstration of enterprise security patterns and modern web technologies. It showcases how Zero-Trust principles can be implemented across distributed applications focused on attribute based access controls (ABAC). Take a look at some of the technologies below for more information on how they are used in this project and my experience working with them.

Docker

Lightweight, portable, and self-sufficient - Docker containers include an application's code, dependencies, and all necessary libraries and configurations to run it reliably across different computing environments.

Kubernetes

A portable, extensible, open source platform for managing containerized workloads and services, that facilitates both declarative configuration and automation.

Helm

Helm addresses the complexity of managing and deploying applications in Kubernetes by providing a structured and reusable way to package and deploy Kubernetes resources.

Keycloak

Keycloak is an open source Identity Provider (ICAM) enabling single sign-on and attribute-based access control (ABAC). Keycloak supports integration with applications through OIDC/OAuth2 and SAML

Hashicorp Vault

Securely manage, store, and access secrets like passwords, API keys, and certificates, as well as other sensitive data. Vault centralizes secrets management, provides on-demand credential generation, supports encryption for data both at rest and in transit.

External Secrets

Securely synchronize secrets from external secret management systems to keep sensitive information like passwords, API keys, and certificates out of version control systems intended for code.

Terraform

An open-source tool that manages infrastructure using Infrastructure as Code (IaC) principles, allowing users to define, provision, and manage cloud and on-premises infrastructure through configuration files written in its declarative language.

Ansible

A software development approach that extends the principles of DevOps to include security as a shared responsibility across development, security, and operations teams.

Packer

A software development approach that extends the principles of DevOps to include security as a shared responsibility across development, security, and operations teams.

GitLab

A comprehensive platform designed for DevSecOps. It functions as a single, integrated platform for version control, collaboration, and continuous integration and continuous delivery (CI/CD).

Git

An open-source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. It allows developers to track changes, collaborate on code, and manage project history.

FluxCD

FluxCD facilitates GitOps by automating the process of synchronizing the state of a Kubernetes cluster with the desired state in Git, ensuring consistency and enabling traceable, auditable orchestration.

Cloudflare

Provides an authoritative DNS hosted zone service that manages a domain's DNS records with built-in security protections. Through its Zero Trust platform, Cloudflare offers secure access control to applications and networks.

Nginx

An reverse proxy server that intercepts incoming client requests, directs them to the appropriate back-end application or web server, and then returns the server's response to the client.

Istio

Istio is a service mesh for Kubernetes, an infrastructure layer that gives applications capabilities like zero-trust security, observability, and advanced traffic management, without code changes.

Firewalls

A security guard for a network, monitoring incoming and outgoing traffic - blocking anything that doesn't meet security rules. Firewalls create a protective barrier against cyber threats like malware, viruses, and unauthorized users.

Networks

Network segmentation and routing helps to contain threats by limiting lateral movement, enhance performance by reducing congestion, improve management by localizing issues, and meet compliance requirements by isolating sensitive data.

Virtualization

A technology that creates software-based, or "virtual," versions of computer hardware, allowing a single physical machine to act as multiple independent systems, such as virtual machines (VMs), servers, networks, or storage devices.

Open WebUI

An extensible, feature-rich, and user-friendly self-hosted AI platform designed to operate entirely offline. It supports OpenAI-compatible APIs, with built-in inference engine for RAG.

LM Studio

Run local AI models like gpt-oss, Qwen, Gemma, DeepSeek and many more on your computer, privately and for free.

Talos

Talos OS (or Talos Linux) is a minimal, immutable, and API-managed Linux operating system designed specifically to host Kubernetes clusters locally or in the cloud.

Nextcloud

Nextcloud is an open-source, self-hosted cloud platform for file storage, sharing, and collaboration, offering a privacy-focused alternative to commercial services like Google Drive or Microsoft OneDrive.

Rocket.Chat

An open-source communication and collaboration platform that offers secure, customizable channels for team communication, with a focus on enterprise-grade features and deployment flexibility.

Jitsi

A collection of free, open-source, and multiplatform applications for voice and video conferencing. Jitsi enables users to conduct secure video calls, share screens, and collaborate remotely
